# kmo-prank

This is a simple script that shows an image on an Ubuntu machine for 4 seconds.

## Requirements
- Ubuntu
- feh installed on the target machine

## Usage

1. Clone the repo:
   git clone https://github.com/MRmacro118/kmo-prank

2. Enter the folder:
   cd kmo-prank

3. Make the script executable:
   chmod +x popup.sh

4. Run the script through SSH or locally:
   ./popup.sh image.png

Replace `image.png` with any image stored in the repo.

The image will appear fullscreen for 4 seconds and then close automatically.
